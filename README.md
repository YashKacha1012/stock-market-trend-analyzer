# Stock Market Trend Analyzer

## 📊 Overview
This project analyzes stock market data using Python, Pandas, and SQL.

## 🚀 Features
- Stock data extraction using yfinance
- Data storage in MySQL
- Exploratory Data Analysis (EDA)
- Monthly trend analysis
- Moving averages (MA20, MA50)
- Volatility analysis

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- MySQL
- Matplotlib
- yfinance

## 📈 Key Insights
- Identified stock trends over time
- Calculated moving averages to smooth price fluctuations
- Analyzed volatility and price ranges
- Compared monthly performance of stocks

## 📂 Files
- `stock_analysis.ipynb` → Data analysis
- `mysql_queries.sql` → SQL queries

## 🌐 Data Source
- Yahoo Finance API (yfinance)
